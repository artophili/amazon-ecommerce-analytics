# Amazon India E-Commerce Analytics

An analytics framework designed to transform raw e-commerce operational logs into business intelligence. This project engineers a Python-driven programmatic data cleaning, relational database (SQL), and interactive visualization—to minimize operational margin leaks and optimize fulfillment networks.
